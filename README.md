## ğŸš€ Deployment

This static website can be easily deployed to various static hosting services.

### GitHub Pages
1.  Ensure your repository is public.
2.  Go to your repository settings on GitHub.
3.  Navigate to the "Pages" section.
4.  Under "Build and deployment", select `Deploy from a branch` and choose `main` as your branch and `/ (root)` as your folder.
5.  Save your changes. Your portfolio will be accessible at `https://[YourUsername].github.io/Portafolio-xd-/`.
